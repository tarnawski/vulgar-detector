You can install VulgarDetector as Wordpress plugin to automatically recognition and flagged as spam comments with vulgar language. VulgarDetector take care your website, so you can focus on more important things than vulgar comments.



BIG DATABASE

The current size of the database allows the effective recognition of vulgar texts. Every month the number of words is growing constantly making applications better and better.

OPEN CODE

Application and Wordpress plugin is completely safe and will not do anything behind your back. All code is open and available on GitHub. For safety, only the database of vulgar words is not publish.

NEW TECHNOLOGY

It's PHP RESTfull application build on the shoulders of Symfony 3 microframework. All data is stored in MySQL database. The project is based on the latest technologies and tools for example Vagrant, Docker, Behat, PHPSpec and more...
